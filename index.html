Heron's Formula

Formula for Triangle Area

Heron's formula calculates the area of a triangle when you know the lengths of all three sides.

Area = √[s(s-a)(s-b)(s-c)]

Where:

  • a, b, c are the lengths of the three sides
  • P = a + b + c is the perimeter
  • s is the semi-perimeter: s = P / 2 = (a + b + c) / 2
  • √ represents the square root function

Python Implementation

# Get side lengths from user
a = float(input("side 1"))
b = float(input("side 2"))
c = float(input("side 3"))
# Step 1: Calculate sum of sides (perimeter)
s = (a + b + c)
print("sum", s)
# Step 2: Calculate semi-perimeter
d = s / 2
print("addition", d)
# Step 3: Calculate s(s-a)(s-b)(s-c)
e = d * (d - a) * (d - b) * (d - c)
print("saikat", e)
# Step 4: Calculate square root for area
f = e ** 0.5
print("area", f)
