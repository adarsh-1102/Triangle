Heron's Formula

Formula for Triangle Area

Heron's formula calculates the area of a triangle when you know the lengths of all three sides.

Area = √[s(s-a)(s-b)(s-c)]

Where:

  • a, b, c are the lengths of the three sides
  • s is the semi-perimeter: s = (a + b + c) / 2
  • √ represents the square root function

Python Implementation

# Get side lengths from user
a = float(input('Enter 1st side:'))
b = float(input('Enter 2nd side:'))
c = float(input('Enter 3rd side:'))
# Calculate semi-perimeter
s = (a + b + c) / 2
# Apply Heron's formula
area_squared = s * (s - a) * (s - b) * (s - c)
area = area_squared ** 0.5
# Display the result
print('Area =', area)
